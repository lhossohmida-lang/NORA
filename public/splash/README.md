# Splash Videos

ضعي ملفّي فيديو الافتتاح هنا:

- `intro-mobile.mp4`  — نسخة عمودية للهاتف (ratio 9:16، حجم ≤ 3MB)
- `intro-desktop.mp4` — نسخة أفقية للحاسوب (ratio 16:9، حجم ≤ 5MB)

كلاهما يُشغَّل تلقائياً، صامتاً، 4 ثوانٍ فقط. إن لم تتوفّر الملفّات،
يعرض `SplashScreen` خلفية الـ ink الافتراضية مع تدرّج لطيف.

اقتراحات للمحتوى:
- لقطات قصيرة لقماش الفساتين، تطريز ذهبي، نسج تقليدي
- ضوء طبيعي، حركة بطيئة (slow-motion)
- بدون صوت — التطبيق يكتم تلقائياً
